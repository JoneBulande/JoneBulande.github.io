Sobre mim

Fullstack por escolha,
mobile por paixão

Sou Jone Bulande, desenvolvedor de software baseado em Maputo. Comecei no mobile com Flutter e evoluí para cobrir todo o stack — de backends em Node.js a websites modernos com HTML, CSS e JavaScript puro.

Tenho apps publicados na Play Store, participei no Code in the Dark 2022 em Maputo, e já entreguei projetos reais para clientes — incluindo o site institucional rhema.co.mz, desenvolvido do zero com UI profissional e animações fluidas.

O que me diferencia: não sou apenas quem escreve código. Penso no produto como um todo — arquitectura, UX, performance e deploy. Entrego soluções completas, não apenas features isoladas.
